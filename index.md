# Marp とは

- Markdown Presentation Ecosystem
  Markdown でプレゼン資料を作成するためのエコシステム
- オープンソースで公開されている

---

<!--
class: content
-->

# Marp の特徴

- CommonMark(Markdown の標準仕様)に準拠
- 拡張シンタックスがサポート
- CSS でスタイリング可能
- HTML, PDF, PowerPoint に出力可能
- 豊富なツールセット

---

<!--
class: content
-->

# 作業の流れ

1. Markdown でスライドを作成
2. VSCode 上で Preview を確認
3. 修正
4. 1 ~ 3 を繰り返す
5. PDF 等に出力

# 良かった点

- CSS の資産を活用できる
- プレゼン資料の配布、再利用が容易
- 発表内容のアウトラインの構成とプレゼン資料の作成がシームレスに繋がる
- 生成 AI の助けを借りれる

---

<!--
class: content
-->

# 難点

- 細かいスタリングが面倒（GUI で操作した方が圧倒的に早い）
- 最低限の CSS の知識が必要
